PROJECT / 3D Animation / Dec 16

I created a short animation through a process involving the complete 3D CG production pipeline of storyboarding, modelling, texturing, UV mapping, rigging, animating, lighting, and rendering. The software used for this project was Maya, and I also had some experience with dynamics and MEL scripting during the process. The resulting animation can be played below.


I also previously had some experience using Blender, to create still frames of computer graphics. This developed my skills in modelling, texturing, UV mapping, lighting, compositing, and rendering, which were useful when moving on to animation.
