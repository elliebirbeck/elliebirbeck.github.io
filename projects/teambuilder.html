PROJECT : Team Builder

Team Builder is a web app developed by myself and two other students over four weeks. It was intended to digitise the approach to creating teams for group projects, replacing the typical pen and paper form submissions and manual allocation.

The application allows admin users to create forms, by adding questions and assigning them weightings based on importance. They can then distribute the form, and other users log in to submit their answers. When enough results have been returned, the admin can run our matching algorithm to automatically build teams of any size. The algorithm uses the question weightings and binary matches of answers between users to create balanced teams with optimised compatibilities.

My role in the team was mainly front-end development, but I also contributed to the database design and added some back-end integration. Tools used in this project include Python, Flask, MongoDB, HTML, CSS, Javascript, JQuery, Bootstrap.
