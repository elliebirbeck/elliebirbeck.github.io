PROJECT : Kaggle Competition

Kaggle is an organisation which hosts machine learning competitions to solve real-world problems, often sponsored by large companies or leading researchers. In March 2016 I took part in one of these competitions, where the aim was to classify types of crime occurring in San Francisco.

Given several hundred thousand pieces of data relating to previous crimes, including the date and time of the occurrence and the latitude and longitude of the location, I experimented with different feature sets and machine learning algorithms to develop a classifier. The algorithms tested were multi-layer perceptrons, support vector machines, naive Bayes, k-nearest neighbour, and random forests.

I extended the given feature set by creating additional spatial metrics such as distances to the city centre and police department districts, grid cell locations, and testing variations in format and type of temporal measurements. Experimenting with different machine learning algorithms and performing ablative analysis to understand the contributions of individual features allowed me to refine the final feature set and build the model. Tools used for this project were Weka, Java, and R.

The competition was judged based on the resulting multi-class logarithmic loss of the classification. My classifier improved on a baseline logloss value of 3.65, reducing it by 28% to 2.61. The winning competition entry achieved 2.05. I hope to tackle another Kaggle competition in the future.
